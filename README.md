# Bilibili点阵显示器
Demo：https://www.bilibili.com/video/av34606451  
  
材料：  
ESP2866  
MAX7219点阵（4联排）  
杜邦线  
MicroUSB线  
3D打印外壳（外观参考上述视频）  
半透明黑色亚克力面板  
螺丝螺母4套  
  
安装步骤简要：  
1、下载Bilibili_LEDMatrix_V1.ino、fonts.h、max7219.h三个文件；  
2、参照注释对Bilibili_LEDMatrix_V1.ino文件中的需要配置的地方进行更改；  
3、按照视频中配置编译程序：https://www.bilibili.com/video/av32725704  ；  
4、接入ESP8266，编译，上传；  
5、完成！
